Standardkonstruktor der Klasse, der die wesentlichen Parameter befüllt.

Parameter
innerRegulationIn dem Parameter wird eine Referenz auf die innerer Regel erwartet, mit der diese Regel einen Regelverbund bilden soll.
effectiveDieser Parameter legt fest, ob die Regel innerhalb des Regelverbundes in Kraft sein soll.
suspendableDieser Parameter legt fest, ob die Regel außer Kraft gesetzt werden darf.
