◆ getParticipantId()

IParticipantId org.dos.tournament.common.player.IParticipant.getParticipantId ( )

Getter-Methode für die Id des Teilnehmers.

Jeder Teilnehmer braucht im Rahmen des Turniers eine eindeutige Id, die für den Teilnehmer steht. Dieses kann z.B. eine Startnummer sein.

Rückgabe
Die Methode liefert die Id im Rückgabewert.
