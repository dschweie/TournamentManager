◆ getCompetitorResult()

IResult org.dos.tournament.common.competition.IEncounter.getCompetitorResult ( IParticipant  competitor)

Die Methode liefert das Ergebnis aus Sicht einer Teilnehmers.

Parameter
competitorIn diesem Parameter wird eine Instanz erwartet, für den das Ergebnis ausgegeben werden soll.
Rückgabe
Die Methode liefert das Ergebnis aus Sicht einer Teilnehmers als Instanz von IResult, falls dieser an der Begegnung teilgenommen hat und ein Ergebnis vorliegt. Andernfalls wird null zurückgegeben.

Die Methode liefert die Information, ob die Begegnung vollständig ist.

Die Methode liefert den Wert false, wenn weitere Teilnehmer zu der Begegnung hinzugefügt werden müssen. Andernfalls wird true zurückgegeben.
