◆ saveParticipant()

boolean org.dos.tournament.common.storage.IStorage.saveParticipant ( IParticipant  participant,
boolean  overwrite 
)

Methode zum Speichern eines allgemeinen Mitpielers.

Parameter
participantIn dem Parameter ist ein Objekt vom Typ IParticipant zu übergeben, dass gespeichtert werden soll.
overwriteIn dem Parameter wird ein Flag erwartet, das signalisiert, ob bestehende Daten überschrieben oder ergänzt werden sollen. Wenn bereits ein Datensatz vorhanden ist, dann kann mit dem Wert false erreicht werden, dass in dem Datensatz nur die neuen Informationen aktualisiert werden. Wenn true übergeben wird, dann wird der Datensatz zunächst gelöscht und nur mit den Informationen gespeichert, die in der Instanz vorhanden sind.
Rückgabe
Die Methode liefert im Rückgabewert die Information, ob das Speichern erfolgreich war.

Implementiert in org.dos.tournament.common.storage.mongo.DatabaseClient.
