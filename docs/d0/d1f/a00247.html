◆ COL_MATCH

final String org.dos.tournament.application.common.panels.components.SuperMeleeMatchdayTable.COL_MATCH = "MATCH"
static

Konstante für den eindeutigen Index der Begegnung im Turnier.

Jede Begegnung soll selbst einen eindeutigen Index haben, der im Turnier nur einmal vorkommt. Damit lassen sich zum Beispiel Begegnungen beschreiben, ohne dass ein konkretes Team zu benennen ist. Es kann z.B. "Sieger aus Begegnung 50" im Spielplan auftauchen und das Team wird erst angegeben, wenn die Begegnung mit Ergebnis versehen ist.

Diese Konstante steht für die Spalte, die nur optional angezeigt wird.
