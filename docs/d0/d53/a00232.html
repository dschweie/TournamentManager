Methode erzeugt die Kopfzeile der Tabelle mit den Texten, die in der GUI angezeigt werden.

In der internen Struktur SuperMeleeMatchdayTable.astrHeader werden nur die Konstanten der jeweiligen Spalten hinterlegt. Mit dieser Methode wird die Kopfzeile der Tabelle mit den Spaltennamen erzeugt, die dann in der GUI dargestellt werden.

Noch zu erledigen:
Die Bezeichner werden aktuell noch nicht aus den Properties gelesen.
Rückgabe
Die Methode liefert die Kopfzeile als Vector zurück.

◆ getColumnIndexOf()

int org.dos.tournament.application.common.panels.components.SuperMeleeMatchdayTable.TableModel.getColumnIndexOf ( String  header)

Getter-Methode liefert den Spaltenindex zu einem speziellen Spaltencode.

In der Tabelle kann es eine variable Anzahl von Spalten geben. Somit kann ein Spalte nicht über einen konstanten Spaltenindex referenziert werden.

Jede Tabelle hat mit dem Vector astrHeader einen Ort, an dem die Spalten dokumentiert sind.

Parameter
headerIn diesem Parameter ist eine Spaltenkennung zu übergeben, zu der der Spaltenindex zurückgegeben werden soll.
Rückgabe
Die Methode liefert den Spaltenindex als int zurück. Wenn die Spalte nicht ermittelt werden kann, da diese nicht eingeblendet oder unbekannt ist, wird der Wert -1 zurückgegeben.

◆ isCellEditable()

boolean org.dos.tournament.application.common.panels.components.SuperMeleeMatchdayTable.TableModel.isCellEditable ( int  row,
int  column 
)

Methode liefert die Information, ob der Zelleninhalt editiert werden darf.

Der Anwender darf in der Tabelle nur die Spalten editieren, in denen Spielergebnnisse der Teams einer Spielpaarung erfasst werden sollen. Alle anderen Zellen sollen nicht geändert werden dürfen.

Parameter
rowIn dem Parameter wird der Zeilenindex der Zelle übergeben, für die angefragt werden soll, ob der Anwender den Inhalt ändern darf.
columnIn dem Parameter wird der Spaltenindex der Zelle übergeben, für die angefragt werden soll, ob der Anwender den Inhalt ändern darf
Rückgabe
Die Methode liefert im Rückgabewert den Wert true, wenn der Anwender den Zelleninhalt bearbeiten darf.

◆ toggleOutputParticipants()

boolean org.dos.tournament.application.common.panels.components.SuperMeleeMatchdayTable.TableModel.toggleOutputParticipants ( )

Mit der Methode wird das Ausgabeformat der Paarungen umgeschaltet.

In der Tabelle werden Spielpaarungen und Spielergebnisse auf der GUI ausgegeben. Für die Paarungen können entweder die Nummern oder Namen der Spieler der Teams angezeigt werden.

Mit dieser Methode kann die Darstellung umgeschaltet werden.

Rückgabe
Die Methode liefert den Wert true, wenn die Darstellung auch Nummern umgeschaltet wurde. Andernfalls wird der Wert false zurückgegeben.
